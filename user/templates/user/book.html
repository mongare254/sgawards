Recognizing and appreciating:

USER LOGIN

Use your username and Password; AD Credentials be validated in the database. If the user credentials are correct you will be redirected to Homepage. If not you will see an error invalid credentials; to mean your user details are incorrect. Click on reset password if you have forgotten password! Click on sign up button if you are a new user

HOME PAGE

A successful login will lead you to the home page, Note that in this page items available are displayed by categories say cars, electronics, watches etc. You can scroll left or down to view more items. Each item is displayed with its name, price tag and description

DECLARE INTEREST IN ITEM

To declare interest in an item you wish to participate in auction, please hover over the item and click the interested button on it. You will then be redirected to another page where you will register to participate in the auction.

REGISTRATION AND PAYMENT

On clicking the Interested button of the item you want to participate in as a bidder, you will be redirected to another page. Please enter the mpesa number you wish to use in the format 2547xxxxxxxxx. A successful payment will lead to successful printing of receipt and an acknowledgement email. The item will be displayed under interested items in the interested items menu.

ADDING AN ITEM FOR AUCTION

A user is allowed to add a new item for auction through the sidebar, Items> add item.The user should duly fill the form provided, and will be redirected with a message item added successfully.The item will also be displayed on the home page.

AUCTION ROOM

A user will be notified when the auction of their item will be done through email and through notifications. In an active auction the user can place their bid using the bid form while the auction room is still active. When it is closed the highest bidder will be declared.

LOG OUT

A user can log out by clicking log out on the side menu
